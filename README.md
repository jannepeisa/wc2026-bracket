# World Cup 2026 — Knockout Bracket Predictor

Interactive single-elimination bracket showing each team's chance of advancing.
Opens to the most-likely bracket; click any team to send them through and the
rest of the bracket recomputes.

Round-of-32 win probabilities come from the de-margined betting market; later
rounds use a calibrated Elo model (opponents there depend on earlier results).

Static page — just open `index.html`.
